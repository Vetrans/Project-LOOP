# 🚀 LOOP AI – AI Customer Feedback Intelligence Platform

An AI-powered Customer Feedback Intelligence Platform developed as part of the **Zidio Development Internship**.

LOOP AI helps organizations collect, manage, analyze, and visualize customer feedback through an intuitive dashboard, enabling data-driven decision making and improving customer experience.

---

## 📖 Overview

LOOP AI is designed to centralize customer feedback and provide meaningful insights through an interactive dashboard. The platform aims to simplify feedback management while supporting future AI-based sentiment analysis and reporting.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/Aayush2024/Zidio-Loop-Project.git
```

## Navigate to Project

```bash
cd project-loop
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```

Application will run at:

```
http://localhost:5173
```
